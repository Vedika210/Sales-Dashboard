# Sales Dashboard
## Key Features
* Interactive Visualizations implemented in Microsoft Power Bi to understand key business metrics.
* Advanced Slicers and Filtering- Filter data by Date, Region, Store Location, Customer Type and Product.
* Central Homepage to navigate to different pages on dashboard.
* Executive view, employee view, finance view and operations view designed.
